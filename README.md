# Kommunelogoar

Eit lite webprogram for å gjere det lettare å finne frem til kommunelogoar.

Prosjektet er byggd med [Create React App (Typescript)](https://github.com/wmonk/create-react-app-typescript).

Datakjelder i botn:

- SSB kommunekjelde.
- Difi logokjelde.
- Pluss litt "himkokt" data.
